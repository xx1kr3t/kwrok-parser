# 🕵️‍♂️ Kwork Projects Monitor Bot

Скрипт на Python для мониторинга биржи фриланса **Kwork.ru**. Бот отслеживает появление новых заказов в реальном времени через внутренний API Kwork (XMLHttpRequest) и мгновенно пересылает их в Telegram-бот с красивым HTML-форматированием.

---

## 🔥 Функционал
* **Прямой парсинг API:** Никакого тяжелого Selenium или ломающегося BeautifulSoup — запросы идут напрямую к JSON-данным Kwork через быстрые HTTP-запросы.
* **Защита от спама при старте:** При первом запуске бот собирает текущие заказы в кэш и не спамит ими, а начинает присылать только *новые* публикации.
* **Память на дубликаты:** Использование `collections.deque(maxlen=500)` гарантирует, что старые заказы не прилетят повторно.
* **Безопасность:** Токен бота и ID чата вынесены в защищенные переменные окружения.

---

## 🛠 Технологический стек
* **Язык:** Python 3.8+
* **Библиотеки:** `requests` (синхронные сетевые запросы), `python-dotenv` (менеджмент конфигурации).
* **Архитектура:** Бесконечный цикл с интервалом опроса 30 секунд и обработкой исключений (`try/except`), чтобы бот не падал при временных сетевых сбоях.
